Course Introduction

In this course, we are going to explore the foundational and advanced concepts of web development, starting with an introduction to the World Wide Web, internet protocols, and markup languages such as HTML and CSS. We will delve into client-side and server-side programming, covering essential JavaScript functionalities, server-side scripting languages, and database management. Additionally, we will work with state-of-the-art frameworks and libraries for both front-end and back-end development, such as React.js, Angular, Ruby on Rails, Flask, and Django. We will also focus on building and deploying web applications, addressing web security concerns, and mastering the techniques for hosting web applications effectively.

Course Outline

  • Client, Server, Communication, Protocols, Ports, Client-Server Architectures, Internet, World Wide Web, HTTP, HTTP Status Codes, Web Clients/Browsers, and Web Servers.

  • JavaScript: Introduction to JavaScript, including basic syntax, variables, data types, statements, operators, conditions, loops, functions, arrays, objects, form validation, DOM manipulation, and asynchronous JavaScript. Also includes using modern JavaScript libraries. Server-side Programming: Introduction to server-side technologies and scripting languages like PHP or Node.js, with database options such as MySQL, MongoDB, or Firebase. Key tasks include: Implementing JavaScript for manipulating HTML, handling attributes, and displaying alerts. Using JavaScript for variables, arithmetic operations, conditionals, loops, functions, arrays, and objects. Handling asynchronous JavaScript operations. Developing and integrating scripts in various locations (head, body, external files). Applying server-side scripting for data handling, form validation, file operations, cookies, sessions, and CRUD operations.

  • The course will explore state-of-the-art front-end frameworks such as Angular or React.js. Key activities include installing and configuring the framework and IDE, setting up the workspace and project environment, implementing core syntax and components, and finally building, testing, and deploying the application.

  • The course covers state-of-the-art server-side technologies like Ruby on Rails or Flask. Key tasks involve installing and configuring the framework and IDE, setting up the project environment, implementing core syntax and components, adding server-side validations and authentication, performing CRUD operations, and building, testing, and deploying the application.

  • The course includes working with Django or another modern framework. Key activities involve installing and configuring the framework and IDE, setting up the project environment, implementing core syntax and components, adding server-side validations and authentication, performing CRUD operations, and building, testing, and deploying the application.

  • The course covers building and hosting web applications, as well as web security. It includes selecting a hosting server and plan, configuring DNS, and uploading and running the website online. Additionally, it introduces web security, including types of threats and prevention measures.
